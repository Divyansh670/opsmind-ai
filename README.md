# OpsMind AI

An AI-powered, agentic code review system that analyzes GitHub Pull Requests for:
- Security vulnerabilities
- Cloud infrastructure cost drift
- Architectural pattern violations

## Stack
- Backend: Go
- Frontend: React + TypeScript
- Database: PostgreSQL + pgvector
- LLM: Free-tier provider (Gemini / Groq / Ollama)
